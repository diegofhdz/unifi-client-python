# unifi-client

A Python client library for the UniFi Network Controller API.

## Installation

```bash
pip install unifi-client
```

## Development

### Setup

```bash
# Clone the repository
git clone https://github.com/diegofhdz/unifi-client-python.git
cd unifi-client-python

# Install in development mode with dev dependencies
pip install -e ".[dev]"
```

### Running Tests

```bash
pytest
```

### Linting

```bash
ruff check .
ruff format .
```

### Type Checking

```bash
mypy src
```
